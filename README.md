# AML/BSA Compliance Intelligence Agent

Automated regulatory monitoring system for large financial institutions.

## Features
- Monitors 13 major regulatory bodies
- AI-powered compliance analysis
- Daily email summaries
- Web dashboard with filtering
- Automatic categorization by regulation type

## Regulators Monitored
- FINRA, FATF, Wolfsberg Group, FinCEN, OFAC
- FRB, FDIC, OCC, SEC, FCA, AMLA, EBA, FFIEC

## Technology Stack
- Python 3.11
- Flask web framework
- Venice AI API for analysis
- Automated email delivery
